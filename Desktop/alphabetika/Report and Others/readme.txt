# Alphabetika

Alphabetika is an interactive Android application designed to teach and engage users with numbers and alphabets. It provides an intuitive interface where users can select items and listen to their corresponding audio pronunciations.

## Features
- **Interactive Learning**: Play audio for numbers and alphabets by selecting them.
- **State Persistence**: Saves the last selected item and restores it when the app restarts.
- **Easy Navigation**: Navigate between different screens with intuitive UI.
- **Media Integration**: Uses `MediaPlayer` for audio playback.

## Technical Details
- Built with **Java** and **Android SDK**.
- Persistent storage using **SharedPreferences**.
- Smooth UI updates with `onClickListener`.
